Roommate-Matching-System

Roommate Matching System Project for OOP class 

Usage

This program requires a student database represented as a comma seperated value which is exported from the google form https://docs.google.com/forms/d/1uKbtDgZSBrIY2WxuDLl8E0MOycVL2qW5Px88rWzcmMs/edit?pli=1. 
A sample database is included in the folder for testing purposes.

This program is compiled using the command:
javac MatchingSystem.java

And is executed using this command:
java MatchingSystem
